# Dark Matter Foundry Importer

Import official fillable Dark Matter 5.5e character sheets into Foundry VTT.

## MVP
Version 0.1 targets Foundry VTT v14 and imports a completed Dark Matter character PDF as a playable dnd5e Actor.

## Development
1. Install Node.js 22 LTS.
2. Run `npm install`.
3. Run `npm test`.
4. Run `npm run build`.

> This project does not redistribute Dark Matter rules text or paid compendium content.
