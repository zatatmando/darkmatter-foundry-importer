# Project Charter

## Goal
Import supported official fillable Dark Matter 5.5e character PDFs into Foundry VTT v14 with minimal manual entry.

## v0.1 Scope
- Read AcroForm fields.
- Normalize core character data.
- Build a dnd5e-compatible character Actor.
- Import one PDF at a time.
- Use custom embedded Items when compendium resolution is unavailable.

## Non-goals for v0.1
- Ship import.
- Batch import.
- Portrait/token import.
- Re-import/update merging.
- Redistribution of paid Dark Matter content.

## Proof of concept
Theron.pdf is the first fixture. The target is a successful playable Actor import.
